VotingBundle
============

A generic voting bundle, purposed to serve as the engine for any voting system.

*WARNING:* This is a prototype and not a final/stable bundle.

Install
-------

    _voting:
      resource: "@RizzaVotingBundle/Resources/config/routing.yml"
      prefix: /voting
